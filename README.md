# Geospatial Analysis - Week 4

This repository contains the materials and exercises for Week 4 of the Geospatial Analysis course.

## Overview

Week 4 focuses on advanced geospatial analysis techniques including:

- Digital Elevation Model (DEM) analysis
- Slope and aspect calculations
- Hillshade generation
- Vector data aggregation
- Spatial interpolation methods

## Requirements

- Python 3.8+
- Jupyter Notebook
- Required packages:
  - `rioxarray`
  - `numpy`
  - `matplotlib`
  - `geopandas`
  - `rasterio`

## Installation

```bash
# Clone the repository
git clone https://github.com/OilSausagee/Geospatial-Analysis.git
cd Geospatial-Analysis

# Switch to Week-4 branch
git checkout Week-4

# Install dependencies
pip install rioxarray numpy matplotlib geopandas rasterio
```

## Usage

1. Open the Jupyter notebooks in the `Lab1/` directory
2. Follow the exercises step by step
3. Results will be saved in the `outputs/` directory
