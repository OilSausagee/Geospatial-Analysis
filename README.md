# Week 2: 避難收容處所空間分析

## 📋 專案概述

本專案進行台灣避難收容處所的空間品質審計與疊圖分析，結合空氣品質監測站資料，評估避難設施的空間分佈與環境品質關係。

## 🗂️ 專案結構

```

├── 📂 data/
│   ├── shelters_cleaned.csv              # 原始避難所資料
│   └── shelters_wgs84.csv              # CRS 轉換後資料
├── 📂 scripts/
│   ├── shelter_aqi_analysis.py           # 主要分析腳本

    ├── shelter_aqi_risk_report.html      # 詳細分析報告
```

## 📊 主要發現

### 🎯 空間品質
- **座標系統一致性**: 資料主要使用 TWD97，需要轉換為 WGS84 進行整合
- **資料完整性**: 98.4% 避難所有效座標，品質良好
- **離群值處理**: 離島地區需要特殊邊界定義

### 🗺️ 空間關係
- **鄰近性優良**: 近 90% AQI 測站在 1 公里內有避難所
- **避難所類型**: 絕大多數為室內設施，適合多種災害類型
- **空間分佈**: 避難所與空氣監測站空間整合度良好

### ⚠️ 技術挑戰
- **座標轉換**: TWD97 到 WGS84 轉換精度影響分析結果
- **離島處理**: 需要擴充台灣邊界定義包含離島
- **資料增強**: 自動分類準確性需要人工驗證

## 🛠️ 技術工具

### Python 套件
- **pandas**: 資料處理與分析
- **folium**: 互動式地圖視覺化
- **matplotlib**: 靜態圖表生成
- **seaborn**: 統計視覺化

### 分析方法
- **Haversine 公式**: 計算地理座標距離
- **空間審計**: 座標品質檢查與離群值偵測
- **關鍵字分類**: 基於設施名稱的自動分類
- **疊圖分析**: 多圖層空間關係視覺化
